## 📖 Sobre

Este repositório contém diversos exercícios resolvidos em **C#** como parte do aprendizado de programação. Cada programa aborda um problema diferente, aplicando conceitos como:

- ✅ Entrada e saída de dados
- ✅ Estruturas condicionais
- ✅ Cálculos matemáticos
- ✅ Orientação a objetos
- ✅ Formatação de resultados

### Pré-requisitos

- [.NET SDK](https://dotnet.microsoft.com/download) (6.0+)
